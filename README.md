This project is a simple conversational chatbot built using **Facebook's BlenderBot** model with the **Hugging Face Transformers** library.

It demonstrates how to load a pretrained Seq2Seq model, tokenize user input, and generate responses using PyTorch.

---

##  Features
- Uses `facebook/blenderbot-400M-distill`
- Built with Hugging Face Transformers
- PyTorch backend
- Easy to extend into an interactive chatbot

---

##  Tech Stack
- Python 3
- Transformers
- PyTorch
